在开始登陆服务器之前，我们先打开命令行ping一下自己服务器上的IP地址是否能ping通

方法：ctrl+r cmd ping ip

ssh root@ip

单线程（1-2个人用） wget -N –no-check-certificate https://softs.loan/Bash/ssr.sh && chmod +x ssr.sh && bash ssr.sh

备用 wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/ssr.sh && chmod +x ssr.sh && bash ssr.sh

  bash ssr.sh

多用户（支持限制流量） wget -N –no-check-certificate https://softs.fun/Bash/ssrmu.sh && chmod +x ssrmu.sh && bash ssrmu.sh

备用  wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubi/doubi/master/ssrmu.sh && chmod +x ssrmu.sh && bash ssrmu.sh

  bash ssrmu.sh
